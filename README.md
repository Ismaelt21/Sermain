# Sermain Web

Landing page corporativa de Sermain, empresa ecuatoriana especializada en mantenimiento, montaje e instalación de maquinaria industrial.

## Desarrollo local

```bash
npm install
npm start
```

La aplicación se abre en `http://localhost:5173`.

## Build de producción

```bash
npm run build
```

El resultado se genera en `dist/`.

## Despliegue

Para servicios como Render, usar:

- Build command: `npm install && npm run build`
- Publish directory: `dist`

La página está construida con React, Vite, Tailwind CSS y Lucide React.
